Valores String

Valores Booleanos: Verdadeiro (true) e Falso (false)

Valores Primitivos: Number e BigInt

Tipo Undefined: Indefinido (variável sem valor)

- Os de agrupamento ()
- Os de atribuição = += -= *= /= %=
- Os de incremento ++ e decremento --
- Os lógicos == === != !== > >= < <=
