Descrição do Trabalho

Com este trabalho busquei utilizar a mudança do estado de vários atributos, através do CSS3, em vários elementos HTML.

Atendendo ao objetivo do desafio 02, o elemento escolhido para apresentacao foi o elemento "iframe".

Com este elemento da para fazer a inserção de conteudos externos e internos. E para esse exemplo, fiz a ancoragem com links de vídeos externos, trazidos do YouTube. Mas, é possivel apresentar páginas dentro de páginas.
