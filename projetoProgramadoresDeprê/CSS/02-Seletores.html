SELETORES

>>> Seletor Universal do CSS: Usa-de o *

>>> Seletor de Tipo de Elemento: Usa-se a própria Tag

Por exemplo, o que está nesta Tag article.

"Neque porro quisquam est qui dolorem ipsum quia dolor sit amet, consectetur, adipisci velit..."

>>> Seletor de ID: Na Tag coloca-se id="nome" e configura no CSS usando o #nomeDoId

>>> Seletor de Classe: Em todas as Tags que irão receber o estilo coloca-se class="nome" e configura no CSS usando .nomeDaClasse

>>> Seletor de Descendente: Menciona o caminho do elemento que irá receber as configurações. Por exemplo até chegar a essa Tag strong.

>>> Seletor Filho: É um conceito de atribuir uma determinada configuração a um único elemento que tenha ligação direta.Exemplo: Este texto está com a Tag em que está dentro da mesma Tag strong, porém não está recebendo a mesma configuração do 1o em, por não ser filha direta. Esta é filha direta da Tag ins que é filha direta da Tag strong.
